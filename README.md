## Run Locally

Clone the project

```bash
  git clone https://github.com/Emin14/armageddon__NextJS.git
```

Go to the project directory

```bash
  cd armageddon__NextJS
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Features

- Order of asteroids
- Separate page for each asteroid
- Infinite scroll
